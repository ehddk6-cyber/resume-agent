# QUESTION TYPE FALLBACK TASK
아래 자기소개서 문항을 TYPE_A~TYPE_I 중 하나로 재분류하라.

# TYPE GUIDE
- TYPE_A: 지원동기/직무 적합성
- TYPE_B: 직무역량/강점
- TYPE_C: 협업/갈등/조정
- TYPE_D: 성장/학습/개선
- TYPE_E: 입사 후 포부/기여
- TYPE_F: 원칙/가치관/신뢰
- TYPE_G: 실패/위기/복기
- TYPE_H: 고객/민원/서비스
- TYPE_I: 우선순위/판단/압박 상황

# Application Target

## Organization
- Company: 농·축협
- Team: 일반관리직 / 금융·경제·교육지원 연계 실무
- Role: 신규직원

## Goal
- 2026년 상반기 농·축협 신규직원 자기소개서 보완
- 기존 초안의 강점은 유지하되, 농협의 복합 사업 구조와 조합원 관점 연결을 더 촘촘하게 정리

## Writing Constraints
- Target language: 한국어
- Tone: 실무형, 신뢰형, 과장 없는 협동조합 조직 적합 문체
- Max length: 문항당 600자 내외

## Questions
1. 성장 가능성을 가장 잘 보여주는 사례와 농협 업무 활용
2. 판단이 쉽지 않은 상황에서 의사결정을 내린 경험
3. 역할 수행 과정에서 신뢰를 얻게 된 경험
4. 중요하게 생각하는 가치/원칙과 수행하고 싶은 역할
5. 교육지원·경제·금융의 복합 구조 경쟁력과 본인의 기여

## Role-specific signals to emphasize
- 데이터 기반 실무 개선 능력
- 공정한 판단과 현장 확인 습관
- 조직 내 조율과 협업형 신뢰 형성
- 조합원 접점에서 서비스 개선 의견을 구조화하는 역량

# resume-agent 프로젝트 수정 검증 프롬프트

## 검증 요청

아래는 resume-agent 프로젝트에 추가된 8개 모듈입니다. 각 모듈의 코드를 검증하고 문제점을 지적해주세요.

## 검증 대상 파일

### Phase 1: 안정성 강화

#### 1. checkpoint.py
**목적:** 파이프라인 중간 실패 시 특정 단계부터 재시작 가능

**검증 항목:**
- [ ] JSON 저장/로드가 정상적으로 동작하는가?
- [ ] 체크포인트 디렉토리 생성이 올바른가?
- [ ] get_resume_point()가 올바른 순서를 반환하는가?
- [ ] 에러 처리가 적절한가?

#### 2. cli_tool_manager.py
**목적:** 사용자가 Codex, Claude, Gemini, Cline 중 선택하여 사용

**검증 항목:**
- [ ] subprocess 호출이 안전한가?
- [ ] 타임아웃 처리가 올바른가?
- [ ] 지원하지 않는 도구에 대한 에러 메시지가 명확한가?
- [ ] execute_with_file()이 파일을 올바르게 읽는가?

#### 3. validators.py
**목적:** 경험 데이터의 STAR 구조, 구체성, 클리셰 검증

**검증 항목:**
- [ ] ValidationResult가 올바르게 동작하는가?
- [ ] STAR 길이 검증 기준이 적절한가?
- [ ] 클리셰 패턴 목록이 충분한가?
- [ ] 증거 레벨 검증 로직이 올바른가?

### Phase 2: 사용자 경험 개선

#### 4. interactive.py
**목적:** 대화형 코칭 세션 (실시간 제안, undo/redo)

**검증 항목:**
- [ ] undo/redo가 deep copy를 사용하여 올바르게 동작하는가?
- [ ] input() 호출이 EOF 에러를 처리하는가?
- [ ] 경험 선택 시 인덱스 검증이 충분한가?
- [ ] 히스토리 크기 제한이 필요한가?

#### 5. progress.py
**목적:** 프로그레스바 및 상태 표시

**검증 항목:**
- [ ] ETA 계산이 division by zero를 방지하는가?
- [ ] 컨텍스트 매니저가 예외 시에도 올바르게 동작하는가?
- [ ] 터미널 폭에 맞게 출력되는가?
- [ ] 유니코드 문자가 모든 터미널에서 지원되는가?

### Phase 3: 지능형 기능 확장

#### 6. vector_store.py
**목적:** 벡터 임베딩 기반 의미적 유사도 검색

**검증 항목:**
- [ ] _simple_embed()가 빈 텍스트를 처리하는가?
- [ ] 코사인 유사도 계산이 0 나누기를 방지하는가?
- [ ] JSON 저장 시 한글 인코딩이 올바른가?
- [ ] 검색 결과가 유사도 기준으로 정렬되는가?

#### 7. quality_evaluator.py
**목적:** 자기소개서 품질 자동 평가 (5개 차원)

**검증 항목:**
- [ ] 가중 평균 계산이 올바른가?
- [ ] 정규표현식이 한글 문장을 올바르게 분리하는가?
- [ ] 점수 범위(0-100)를 초과하지 않는가?
- [ ] 피드백 생성이 중복되지 않는가?

#### 8. feedback_learner.py
**목적:** 사용자 피드백 기반 패턴 학습 및 추천

**검증 항목:**
- [ ] Thread-safe한가? (동시 접근 시)
- [ ] 평점 계산 시 0 나누기를 방지하는가?
- [ ] 리포트 내보내기 시 파일 권한 문제를 처리하는가?
- [ ] 개선 영역 식별 로직이 적절한가?

# Review Draft

You are reviewing an application draft for quality and safety.

## Inputs
- Draft: `artifacts/writer.md`
- Candidate facts: `profile/facts.md`
- Experience bank: `profile/experience_bank.md`
- Target brief: `profile/targets/example_target.md`

## Task
Review the draft for:
- factual overreach
- weak evidence
- vague claims
- failure to answer the prompt
- tone mismatch
- likely interview risk

## Output sections
- Verdict
- Major issues
- Minor issues
- Suggested rewrites
- Clean revised version

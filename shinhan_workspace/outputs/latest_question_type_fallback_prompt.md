# QUESTION TYPE FALLBACK TASK
아래 자기소개서 문항을 TYPE_A~TYPE_I 중 하나로 재분류하라.
반드시 JSON 배열만 출력한다.
형식: [{"question_id":"q1","question_type":"TYPE_A","reason":"짧은 근거"}]

# TYPE GUIDE
- TYPE_A: 지원동기/직무 적합성
- TYPE_B: 직무역량/강점
- TYPE_C: 협업/갈등/조정
- TYPE_D: 성장/학습/개선
- TYPE_E: 입사 후 포부/기여
- TYPE_F: 원칙/가치관/신뢰
- TYPE_G: 실패/위기/복기
- TYPE_H: 고객/민원/서비스
- TYPE_I: 우선순위/판단/압박 상황

# QUESTIONS
- q1_strength: 다른 지원자와 차별화되는 본인만의 강점을 한가지 선택하여, 이를 신한은행에서 어떻게 발휘할 수 있을지 작성해 주세요.
- q4_future_vision: 은행 입행 10년 혹은 20년 후 본인의 모습을 상상해보고 어떤 분야의 전문가로 성장하고 싶은지 기술해 주세요.

# Education Strategy Notes

## 목적
- 학교교육 전략 메모를 프로파일과 분리해, 기관별 지원 프로젝트에서만 쓰는 참고 자료로 유지

## 왜 프로파일에 넣지 않는가
- `공단 학교교육.txt`, `교과목 가점.txt`는 개인 사실보다는 지원 전략 문서에 가깝습니다.
- 과목 자체는 사실이지만, 어떤 과목을 어느 기관에 어떻게 배치할지는 지원 기관별로 달라집니다.
- 따라서 기본 프로파일에 고정하면 오히려 특정 기관 기준 전략이 섞일 수 있습니다.

## 어떻게 쓰는가
- 공고별 프로젝트를 만들 때 교육사항 입력 초안으로 사용
- NCS 코드, 직무 키워드, 교육사항 문항 대응 시 참고
- `question_map` 또는 기관별 `project` 메모를 만들 때만 인용

## 현재 확인한 전략 방향
- 국민연금공단: 법·행정, 사회·복지, 인문·기타를 고르게 배치하는 전략
- 국민건강보험공단: 사회조사방법론, 심리학의이해, 행정법2, 행정조직론, 사회복지정책론 중심 전략
- 경영·경제 계열 문항에서는 조사방법론, 행정계량분석, 리더십과커뮤니케이션의 활용 가능성 높음

## 실무 규칙
- 과목명은 성적증명서 원문 기준으로 입력
- 기관 예시표에 억지로 맞추기보다, 실제 과목명과 설명 가능성을 우선
- 학점보다 직무 연결성과 설명 가능성을 먼저 고려
- 교육사항은 자소서 본문보다 면접에서 "이 과목에서 무엇을 배웠는가"까지 답변 가능해야 함
